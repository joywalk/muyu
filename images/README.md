# 图片资源目录

## 需要添加的文件

### fish.png（必需）
- **用途**：木鱼图标
- **建议尺寸**：200x200px
- **格式**：PNG（推荐透明背景）

- 背景透明，便于适配不同背景色
- 图标风格：简约、现代、禅意
- 颜色：白色主体，可带渐变
